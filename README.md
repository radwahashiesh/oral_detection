# Oral Lesion Detection Web Application

A professional web application for AI-powered oral lesion detection and classification using hybrid deep learning models.

## Features

- **Hybrid AI Model**: Combines image analysis with clinical description processing
- **12 Lesion Types**: Comprehensive oral pathology coverage
- **Professional UI**: Medical-grade interface designed for healthcare professionals
- **Real-time Analysis**: Instant predictions with confidence scores
- **Differential Diagnosis**: Top 3 predictions with probability scores
- **REST API**: Programmatic access for integration
- **Mobile Responsive**: Works on all devices

## Supported Lesion Types

1. Adenomatoid Odontogenic Tumor
2. Ameloblastic Fibroma
3. Calcifying Odontogenic Cyst
4. Glandular Odontogenic Cyst
5. Odontoma (Complex)
6. Odontoma (Compound)
7. Ameloblastoma (Plexiform)
8. Odontogenic Keratocyst
9. Radicular Cyst
10. Calcifying Epithelial Odontogenic Tumor
11. Odontogenic Myxoma
12. Squamous Odontogenic Tumor
